How to use it:

  1. Create a budget
  2. Add your accounts - an actual places where you have the money (e.g. bank accounts, investment platform, physical wallet)
  3. Add your funds - money allocated from your accounts to spending categories (e.g. housing, utilities, fund), certain type of savings or irregular expenses (e.g. backup savings, vacation, new car)
  4. Add income - choose an account where you received the money, and a fund to which you want to allocate it
  5. Transfer your funds - when you decide to change your money allocation
  6. Add new expense - choose account from which you paid and fund from which to deduct spent amount

How it works:

  • Each account can have only one currency. For example if you have multiple currencies in your bank account, you'll need to create separate accounts for each currency.
  • Funds can have multiple currencies, so the balance of a fund is represented by a dictionary with currencies as keys.
  • Budget operation date can be set in the future.
  • Only budget operations with date in the future can be removed.
