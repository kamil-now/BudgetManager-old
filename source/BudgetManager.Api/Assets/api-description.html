How to use it:

  1. Create a budget
  2. Add your accounts - an actual places where you have the money (e.g. bank accounts, investment platform, physical wallet)
  3. Add your spending categories - for you regular expenses
  4. Add your funds - distribution of money from your accounts, each dedicated for a certain type of savings or irregular expenses (e.g. backup savings, vacation, new car)
  5. Add income - choose account where you received the money - it will create unallocated funds
  6. Allocate new income to your funds
  7. Transfer your funds - when you decide to change your money allocation
  8. Add new expense - choose account from which you paid and fund from which to deduct spent amount

How it works:

  • Each account can have only one currency. For example if you have multiple currencies in your bank account, you'll need to create separate accounts for each currency.
  • Funds can have multiple currencies, so the balance of a fund is represented by a dictionary with currencies as keys.
  • Spending Fund is the default fund of a budget and cannot be deleted.
  • If the expense does not have the FundId specified, it will be automatically set to Spending Fund ID.
  • Spending Fund categories are optional.
  • Budget operation date can be set in the future.
  • Only budget operations with date in the future can be removed.
